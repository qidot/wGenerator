使用步骤
如果你的电脑中安装了多个jdk的版本,脚本里面是支持按照指定的jdk脚本执行的哦！

- 下载压缩包文件
- 解压缩到本地目录
- 重命名dev.properties => app.properties
- 设定您的数据库连接信息
- 安装JDK1.8 (必备,如果没有请先安装)
- 安装pandoc

##mac系统使用如下命令安装

brew install pandoc

##windows下请下载安装包

https://github.com/jgm/pandoc/releases/tag/2.1.1
- mac系统运行 run.sh（注意文件执行权限 chmod +x run.sh）
- win系统运行 run.bat
命令行参数介绍
序号 参数 说明
1 -c 用于设定使用哪个数据库连接,不指定则默认选择的是default
2 -d 用于设定使用哪个数据库,如不指定则从数据库的连接字符串中解析数据库(此时请在连接字符串中加上数据库名)
3 -t 用于设定表名(与-i参数二选一,必须存在一个,如果2个同时存在优先使用-t参数解析)
4 -i 用于查看数据库结构文档的参数
5 -m 用于设定使用哪个模板进行生成,如不指定则从app.properties中读取设定的值来解析
6 -l 设置运行的语言, -l java
7 -u 用于设定用户信息, -u key字符
运行命令(采用了Markdown模板转换,pandoc直接生成html并打开,优化美观度!)
run.sh
在终端中输入命令来执行脚本:
cd ~/您的存放目录/
sh run.sh -c default -d db1 -t table1 -m template1.vm
run.bat
打开cmd窗口执行命令
cd 盘符:/您的存放目录/
run.bat -c default -d db1 -t table1 -m template1.vm
执行后
